# GoldPrice-Prediction

This project implements a predictive analytics approach to forecast gold prices for the year 2026 using historical gold price data. The dataset includes yearly gold prices (in USD and INR) along with exchange rates.

The workflow involves data cleaning, exploratory data analysis, and visualization to understand long-term trends in gold prices. A Linear Regression model is applied to capture the relationship between time (year) and gold price, enabling future predictions.

The project also includes:

Data preprocessing and handling missing values
Trend visualization using graphs
Model training and prediction
A demonstration script for predicting future prices
A detailed report explaining methodology, results, and limitations

This project highlights how simple machine learning models can be effectively used for time-based forecasting and financial trend analysis.
